A Turbowarp Mod that is primarily focused on Scripting with multiple code aspects and languages! can use and help use HTML, CSS, JS, TSX, TS, Python and APIs. Click https://gist.github.com/mohib-7428/92961b44f1795b3abe0ad02b588c1f4b to see the original file and the update logs (in comments)

Tips:
1: Text may not show properly, in that case, use this CSS script with your HTML: p { color: black !important }.

2: Your Previous HTML elements do NOT get deleted when the project is stopped, if you want to clear the elements on flag clicked then use the when green flag clicked block and the clear all overlays block underneath the hat block.

3: By default, the HTML layer sits on top of the Scratch stage, this can cause sprite to be unclickable or unaccessible, in that case, press the toggle HTML overlays button and set it to Hide then click it again and the elements will disappear, you can always bring them back by toggling the dropdown to show.
